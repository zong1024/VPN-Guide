## 第二步：导入节点

**安卓**（ex：nekobox）：右上角点击最顶上，然后点击从剪贴板导入链接，导入完成后选中节点，点击小飞机按钮即可

**苹果**（ex：v2box）：复制节点链接（步骤同**安卓**）选中配置界面，点击加号，点击从剪贴板导入链接。
